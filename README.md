#  Smart Event Dashboard

A simple web-based event management dashboard built using HTML, CSS, and JavaScript.  
This project allows users to add, view, and delete events dynamically using DOM manipulation.

---

##  Features

-  Add new events  
-  Delete individual events  
-  Clear all events  
-  Add sample events  
-  DOM manipulation demo section  
-  Event details include:
  - Title
  - Date
  - Category
  - Description  

---

##  Technologies Used

- HTML5 – Structure of the webpage  
- CSS3 – Styling and layout  
- JavaScript (Vanilla JS) – Functionality and DOM manipulation

##  How It Works

### 1 Add Event
- User fills the form.
- Clicks "Add Event" or presses Enter.
- Event card is dynamically created and added to the dashboard.

### 2️ Delete Event
- Each event card has a delete (X) button.
- Clicking it removes the specific event.
- If no events remain, an empty message appears.

### 3️ Clear All Events
- Removes all event cards.
- Displays the empty-state message.

### 4️ Add Sample Events
- Predefined sample events are added to the dashboard.
